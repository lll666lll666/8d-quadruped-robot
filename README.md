# 8d-quadruped-robot
基于PuppyPi的八自由度四足机器人控制以及硬件实现

base_driver是基于STM32F405RGT6和PCA9685的舵机驱动模块的腿部逆运动学解算，接收树莓派上位机的足端位置指令控制腿部运动

实现了对PuppyPi四足机器人的足端逆运动学和正运动学解算以及控制
i2c1用于驱动PCA9685芯片
uart4用于与上位机通信，接收树莓派的足端位置指令

Stand_Init()函数用于初始化四足机器人的站立姿态，设置每个舵机的初始位置，使机器人处于稳定的站立状态。
分别在x，z轴进行了三次曲线的插值，生成了平滑的运动轨迹，使机器人能够平稳地过渡到站立姿态。

Debug_Station()用于矫正各个关节电机零位，引入偏置。
